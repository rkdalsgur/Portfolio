보안 자료구조에 대한 수업자료들

포인터

포인터(주소값을 저장하는 변수)
포인터 변수를 이용하여 연결된 주소의 변수 영역을 액세스할 수 있음

주소 연산자: &
변수의 주소를 얻기 위해 사용
변수 앞에&를 사용하여 변수의 주소를 사용할 수 있음
포인터 = &변수;
ex) int I = 10;
int *ptr;
ptr =&i;


포인터 변수를 선언할 때
포인터 변수로 연결할 변수의 선언형과 같은 형태로
선언해주어야 함
예를들어 int i;
를 포인터로 연결시킬 때
int *ptr; (포인터 선언)
ptr = &i; (변수 연결)
로 선언해주어야 함
int I, j;
int *ptr;
ptr = &i;
*ptr = 10;
j = *ptr; /* j에 I에서 저장해온 값인 10을 포인터 변수로 삽입 */
I j 에 포인터로 각각 10을 넣는다고 했을 때
대충 이런식으로 하면 됨.
* 붙이는지 헷갈릴거같아서 미리 적어둠.

포인터는 데이터 값이 아니라 메모리 주소를 다루는 거라
int *ptr = 7 <-이런 형식의 선언은 불가능함
char *ptr = “korea”; - 문자형 포인터에 문자열의 시작 주소를 지정
char A[100];
char *ptr = A; - 배열을 포인터에 시작 주소를 지정

문자열 + @
ex) Dreams come true! + 7 로 출력시키면
8번째 글자부터
come ture!가 출력된다.

문자열값에 더하기나 빼기를 하면 중간부터 출력하거나 거꾸로 출력할수 있음.

2차원 문자열배열을 1차원 포인트계열로 선언 가능
ex) char string [3][10] = {“dreams”,“come”,“true!”};

출력시

dreams
come
true!
char *ptr[3] = {{“dreams”},{“come”},{“true!”}};
이렇게 저장됨

ptr [0] = dreams
ptr [1] = come
ptr [2] = true!
ptr배열 원소의 값을 변경함으로서 문자열 내용 변경 가능

이중 포인터
- 포인터를 가리키고 있는 포인터

사용 예)
char **ptr;
이중 포인터를 사용하면 포인터 배열을 한번에 가져올 수 있음
