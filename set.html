보안 자료구조에 대한 수업자료들

배열

배열 선언과 메모리 할당 구조 예
ex) 40명의 중간고사 점수 배열 선언
int mid_score[40]; ...
4바이트씩 0부터 39까지의 40자리 공간이 만들어짐

1차원 배열의 초기화 형식
자료형 배열이름[배열크기] = {초깃값 리스트};
ex) A[5] = {1,2,3,4,5}
A[ ] = {1,2,3,4,5}
배열 내에 원소만 차 있으면 배열크기 생략 가능
초깃값보다 원소가 덜 들어갈 경우 남은 공간은 0으로 표기
초깃값보다 원소가 더 들어갈 경우 부족한 공간에 있는 원소들은 표기되지 않음.
문자형 선언 (char) = %c
문자형의 경우는 문자 개수랑 배열 크기를 같게 설정해주면 잘 적용됨.
문자열 선언(string) = %s
문자열 선언의 경우 배열을 1칸 더 잡아먹는다
-문자열+마지막에 null \0 값이 추가되기 때문
그래서 배열 크기를 문자열의 개수보다 1칸 더 높게 잡아줘야 한다.
정수형 선언 = %d

다차원 배열
배열의 차수 만큼 [배열크기] 항목을 추가
2차원 배열의 선언 형식
배열이름 [배열크기] [배열크기]
만약 20개의 자료를 5개씩 4줄로 표현하고 싶다고 생각했을 때 사용할 수 있는 배열.
- 행 우선 ex) A[2][3]
[0][0] [0][1] [0][2]
[1][0] [1][1] [1][2]

3차원 배열의 선언 형식 배열이름[배열크기](면)[배열크기](행)[배열크기](열)
- 면 우선 ->행 -> 열 순으로 우선순위 결정
만약 2차원 배열에 값을 넣는다하면
I[2][3] = {1,2,3,4,5,6}또는
I[2][3] = {1,2}{1,2,3} 으로 넣을 수 있음.
